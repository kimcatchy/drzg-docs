    주사위

    • 주사위 던지기
    • 10분 뒤에 만료

    색상추천

    • 지향색을 입력하지 않을 경우 서로 5개 색상의 조합 추천
    • 지향색 최대 2개까지 입력 시 입력한 색과 어울리는 조합 추천
    • 입력한 지향색은 별표로 표시

    램집사

    • 램집사 역할이 없을 경우 새로 생성
    • 이미 있을 경우 기존의 역할을 제거하고 새로 생성
    • 서버 관리자만 사용 가능
